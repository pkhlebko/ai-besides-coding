## Bug Fix: Pagination Off-by-One (#DEV-4421)

QA reported: last page sometimes returns duplicate items from previous page.

Root cause: `OFFSET` calculation was using 0-based page index but the API expected 1-based. The formula `(page - 1) * limit` was being applied before normalizing the input.

```ts
// Bug
const offset = page * limit; // page=1 → offset=limit (skips first page!)

// Fix
const offset = (page - 1) * limit; // page=1 → offset=0 ✓
```

Added regression test covering pages 1, 2, and last page.
