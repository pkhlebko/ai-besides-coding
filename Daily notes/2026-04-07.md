# Remarks

## Hotfix v2.3.1 — Root Cause

Memory spike traced to unbounded in-memory queue in `EventDispatcher`. When downstream service was slow, events accumulated without back-pressure. Fix: added queue size limit (1000) with overflow logging.

```ts
// Before
this.queue.push(event);

// After
if (this.queue.length >= MAX_QUEUE_SIZE) {
  logger.warn('Event queue full, dropping event', { type: event.type });
  return;
}
this.queue.push(event);
```
